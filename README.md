# RPGGameLearn
    欢迎你加入到RPGGameLearn项目。该项目为在Unity3D环境下开发RPG游戏的demo。项目遵守MIT License，
您可以创建pull request，也可以留下您的宝贵意见，也可以clone走进行二次开发。

## 项目使用的工具
###开发工具
* Visual Studio Code
* Unity3D 2018.4.6f1
### 美工工具
* Blender 2.8 OR 其它建模工具(*.fbx, *.obj)
* Photoshop OR Gimp
* Illustrator OR Inkscape
### 文档编辑
* drawio
* xmind
* Microsoft Office OR Google Docs
